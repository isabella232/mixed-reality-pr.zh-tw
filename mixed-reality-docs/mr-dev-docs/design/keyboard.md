# <a name="keyboard"></a>鍵盤

![鍵盤](images/UX_Hero_Keyboard.jpg)

您的混合現實應用程式可以隨時叫用系統鍵盤，其行為會根據目標平臺的功能而定。 例如，HoloLens 2 上的鍵盤支援直接的互動，而 HoloLens (第一代) 鍵盤則支援注視和空中點擊。
