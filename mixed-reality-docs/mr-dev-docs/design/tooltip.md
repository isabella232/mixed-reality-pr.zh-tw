# <a name="tooltip"></a>工具提示

![工具提示](images/UX_Hero_Tooltip.jpg)

工具提示是連結至另一個控制項或物件的簡短描述。 它們是用來在仔細檢查時傳達提示或額外的資訊。 工具提示可協助使用者瞭解未直接描述的實體環境中不熟悉的物件。
