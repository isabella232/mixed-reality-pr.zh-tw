# <a name="winrt-in-unreal"></a>Unreal 中的 WinRT

在您的 HoloLens 開發過程中，您可能需要使用 WinRT 來撰寫功能。 例如，在 HoloLens 應用程式中開啟檔案對話方塊，需要 FileSavePicker 在 winrt 標頭檔中。 從4.26 版開始，Unreal 的組建系統支援 WinRT。
