# <a name="azure-spatial-anchors"></a>[Azure Spatial Anchors](#tab/asa)

Azure 空間錨點 (ASA) 是一種跨平台服務，可讓您建置空間感知的混合實境應用程式。 您可以使用 Azure 空間錨點，在全球規模的多個裝置之間對應、保存及共用全像攝影內容。
