1) <span data-ttu-id="fca29-187">判斷應繪製哪些頂點，以及它們在螢幕空間 (頂點著色器的位置) </span><span class="sxs-lookup"><span data-stu-id="fca29-187">Determining which vertices should be drawn and where they are in screen space (the Vertex shader)</span></span>
    - <span data-ttu-id="fca29-188">每個網格的頂點著色器都會針對每個頂點執行。</span><span class="sxs-lookup"><span data-stu-id="fca29-188">The Vertex shader is executed per vertex for every mesh.</span></span>

<span data-ttu-id="fca29-191">一般而言，著色器會進行許多轉換和光源計算。</span><span class="sxs-lookup"><span data-stu-id="fca29-191">Typically, shaders do many transformations and lighting calculations.</span></span> <span data-ttu-id="fca29-192">雖然複雜的光源模型、陰影和其他作業可以產生絕佳的結果，但它們也有價格。</span><span class="sxs-lookup"><span data-stu-id="fca29-192">Although complex lighting models, shadows, and other operations can generate fantastic results, they also come with a price.</span></span> <span data-ttu-id="fca29-193">減少著色器中計算的作業數目，可以大幅減少每個畫面的 GPU 所需的工作量。</span><span class="sxs-lookup"><span data-stu-id="fca29-193">Reducing the number of operations computed in shaders can greatly reduce the work needed for the GPU per frame.</span></span>

##### <a name="shader-coding-recommendations"></a><span data-ttu-id="fca29-194">著色器編碼建議</span><span class="sxs-lookup"><span data-stu-id="fca29-194">Shader coding recommendations</span></span>

    - <span data-ttu-id="fca29-199">一般來說，頂點的數目小於圖元數 (720p 是921600圖元、1080p 是2073600圖元，依此類推) </span><span class="sxs-lookup"><span data-stu-id="fca29-199">Generally, the number of vertices is much smaller than the number of pixels (720p is 921,600 pixels, 1080p is 2,073,600 pixels, and so on)</span></span>

#### <a name="remove-gpu-stages"></a><span data-ttu-id="fca29-200">移除 GPU 階段</span><span class="sxs-lookup"><span data-stu-id="fca29-200">Remove GPU stages</span></span>

<span data-ttu-id="fca29-201">後續處理的效果可能很昂貴，而且會提高應用程式的填滿率，包括如 MSAA 的消除鋸齒技術。</span><span class="sxs-lookup"><span data-stu-id="fca29-201">Post-processing effects can be expensive and increase the fill rate of your application, including anti-aliasing techniques like MSAA.</span></span> <span data-ttu-id="fca29-202">在 HoloLens 上，建議您避免使用這些技術和其他著色器階段，例如幾何、輪廓和計算著色器。</span><span class="sxs-lookup"><span data-stu-id="fca29-202">On HoloLens, we recommended avoiding these techniques and additional shader stages such as geometry, hull, and compute shaders.</span></span>

## <a name="memory-recommendations"></a><span data-ttu-id="fca29-203">記憶體建議</span><span class="sxs-lookup"><span data-stu-id="fca29-203">Memory recommendations</span></span>

#### <a name="object-pooling"></a><span data-ttu-id="fca29-206">物件集區</span><span class="sxs-lookup"><span data-stu-id="fca29-206">Object pooling</span></span>
